Obtaining the code¶

The source code for django-email-auth is tracked using the Git version control system, and is hosted on GitHub at the URL http://github.com/blairbonnett/django-email-auth/.

Development version¶

In general, the development version should be perfectly usable, although it may contain bugs. It can be obtaining by cloning the repository located at git://github.com/blairbonnett/django-email-auth.git with Git. If you use the Git command line interface, this can be done with the command:

git clone git://github.com/blairbonnett/django-email-auth.git

Alternatively, you can download the latest version of the code here.
